Op deze website ga ik je op de hoogte houden hoe het project voor Embedded Systems loopt.
Als project heb ik gekozen om een wekker te maken. Deze wekker connecteert via wifi met je eigen Google-Calendar. Via deze connectie zet de wekker automatisch een alarm voor al je afspraken.

Ben je soms te laat op je afspraak? Deze wekker houdt ook rekening met het verkeer zodat je in de toekomst altijd op tijd bent!
